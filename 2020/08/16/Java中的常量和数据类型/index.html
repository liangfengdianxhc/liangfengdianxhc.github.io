常量

概述

常量：是指在Java程序运行期间固定不变的数据。

分类

常量的打印输出

注意：
                1、字符常量的打印输出只能用单引号并且单引号中间必须有且只有一个字符（空格也算）
                2、字符串常量打印输出时双引号中间可以什么都不输入
                3、空常量不能使用 System.out.println() 方法直接打印输出

变量和数据类型

数据类型

数据类型的分类

Java的数据类型分为两大类：

  • 基本数据类型：包括整数、浮点数、字符、布尔
  • 引用数据类型：包括类、数组、接口等

注意：除了基本数据类型以外都是引用数据类型

基本数据类型

四类八种基本数据类型

注意：
  1、字符串不是基本数据类型。
  2、浮点型可能是个近似值并非精确值
  3、数据范围与所占用的内存字节数不一定相关，例如float数据范围比long的数据范围更加广泛，但是float占用内存4字节而long占用内存8字节
  4、整数和浮点数都有默认类型，当我们想使用别的数据类型时要加上想使用的数据类型的大写首字母为后缀
